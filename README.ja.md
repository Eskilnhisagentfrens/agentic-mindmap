# Agentic Mindmap

[English](./README.md) · [中文](./README.zh.md) · **日本語**

LLM エージェントと連携できる、ローカル動作の XMind 風マインドマップ。デスクトップ（macOS Electron）とブラウザの両方で動き、**データは完全にあなたのマシン内にとどまります**。

> 🤖 **AI 展開**：任意のノードを選択し、ツールバーの 🤖 ボタンを押すと、DeepSeek-reasoner がそのノードに対して 3〜5 個の妥当な子ノード（サブタスク・サブトピック・要点）を生成します。子ノードは親ノードの言語を自動で踏襲し、既存の子ノードと重複しないように配慮されます。

## 起動

### macOS デスクトップ版（推奨）

```bash
git clone https://github.com/Eskilnhisagentfrens/agentic-mindmap.git
cd agentic-mindmap
npm install      # 初回のみ
npm start        # 起動
```

`.dmg` をビルド：

```bash
npm run dist     # 出力先は dist/
```

### ブラウザ版

`index.html` をダブルクリック、または：

```bash
open index.html
```

スマホで見たい場合は AirDrop / iCloud Drive で `index.html` を端末に送り、Safari で開いてください。iPhone 17 Pro のダイナミックアイランドとノッチに最適化されています。

## AI 展開のセットアップ

🤖 ボタンを使うには API キーが必要です — DeepSeek（Anthropic より約 30 倍安く、推奨）か Anthropic を選べます。

**方法 A：macOS Keychain（推奨。シェル履歴に痕跡を残しません）**

```bash
security add-generic-password -a "$USER" -s "DEEPSEEK_API_KEY" -w 'sk-...'
```

**方法 B：環境変数（`~/.zshrc` などに記述）**

```bash
export DEEPSEEK_API_KEY="$(security find-generic-password -a "$USER" -s DEEPSEEK_API_KEY -w)"
# Anthropic を使う場合：
export ANTHROPIC_API_KEY=sk-ant-...
```

アプリは Electron メインプロセス内で**遅延ロード**でキーを読み込みます。`localStorage`、レンダラのメモリ、シェル履歴のいずれにも到達しません。詳しいセキュリティモデルは [docs/architecture.md](./docs/architecture.md) を参照してください。

### マウス / タッチ

- **ノードのドラッグ**：
  - 他ノードの中央にドロップ → 子ノードになる
  - 他ノードの上端 / 下端にドロップ → 前 / 後ろの兄弟になる
  - 空白にドロップ → 自由位置（緑の点線枠が出て、サブツリーごと移動）
  - ドラッグ中に `Esc` でキャンセル
- **ノードのリサイズ**（選択するとハンドル表示）：
  - 右下の丸印 → 幅と高さを自由に変更
  - 右辺 → 幅のみ（テキスト自動折返し）
  - 下辺 → 高さのみ
  - `Shift` + 右下 → 比率維持
- **キャンバス**：
  - スクロール / 2 本指トラックパッド → パン
  - `⌘` + スクロール / ピンチ → ズーム
  - 空白部分のドラッグ → キャンバスをパン

### iPhone

- タップで選択、ダブルタップ / 長押しで編集
- 下部ツールバー：子追加、兄弟追加、アイコン、色、メモ、削除
- ピンチでズーム、シングルタッチでパン
- それ以外は上部ツールバーから

## 高度な機能

### 色とアイコン

ツールバー 🎨 で 12 色パレットからノードに色を付けます。**子ノードは最も近い色付き祖先の色を継承**し、エッジの色も枝に追従します。子ノードに別の色を付けたい場合は明示的に色を指定すると上書きされます。

ツールバー 😀 で絵文字アイコン（24 個）を付けられます。

### メモ

ツールバー 📝 で右下にメモパネルが開きます。複数行対応。メモが付いているノードは右上に小さな青い点が表示されます。

### リレーション 🔗

親子関係に依存しない、ノード間のクロス接続です。

1. 始点ノードを選択 → 🔗 をクリック
2. ターゲットノードをクリックして接続成立
3. 線またはラベルをダブルクリック → テキスト編集
4. シングルクリックで選択、`Delete` で削除
5. 作成中に `Esc` でキャンセル

### サマリー 📎

兄弟ノードのグループに大括弧と要約テキストを追加します。

1. ノードを選択 → 📎 をクリック（親ノード上にサマリーが追加される）
2. `Shift` + 兄弟クリックで範囲を拡張
3. ラベルをダブルクリックで編集
4. 選択状態で `Delete` で削除

### 境界 ⬚

枝を点線の色付き枠で囲んでグループ化します。ノードを選択 → ⬚ でトグル。色はデフォルトでノードの色を踏襲します。

### アウトライン 📋

サイドパネルにツリー全体を表示：
- 行クリックでキャンバス上のノードに飛んでセンタリング
- ダブルクリックでテキスト編集
- 三角アイコンで折りたたみ / 展開

キャンバスと双方向で同期します。

### レイアウト

ツールバー 🗺️ / 🌳 でトグル：
- **放射状**（デフォルト）：ルートノードが中央、子ノードが左右に展開
- **右ツリー**：ルートが左、ツリーが右に展開

### 検索

`⌘F` で検索ボックスを開き、ノードのテキストとメモをライブマッチング。マッチは光って強調、それ以外はフェード。`Enter` / `⇧Enter` で循環、`Esc` で閉じます。ジャンプ時に折りたたまれた祖先を自動で展開し、見える位置までキャンバスをパンします。

## ファイル形式

### JSON（無損失）

デフォルトのソース形式。すべてのデータを保持：構造、アイコン、色、メモ、サイズ、位置オフセット、リレーション、サマリー、境界。

`⌘S` またはツールバー 💾 で保存。

### Markdown（汎用）

階層をネスト箇条書きで表現。Obsidian / Typora / GitHub と互換。テキスト、アイコン、色、メモを保持しますが、**リレーション / サマリー / 境界 / サイズ情報は出力されません**（汎用形式を保つため）。

```markdown
# 🎯 ルート <!-- c:#89b4fa -->

- 💡 ブランチ A <!-- c:#f9e2af -->
  > これはメモです
  - アイデア 1
  - アイデア 2
- ⭐ ブランチ B
```

ルール：
- `#` 始まりの行 → ルート
- 半角スペース 2 個下げの `-` / `*` / `+` → 子ノード
- 行頭の絵文字 → ノードアイコン
- 行末の HTML コメント `<!-- c:#hex -->` → ノード色
- `> ` 始まりの行 → ノードメモ

`⌘⇧E` またはツールバー 📤 で書き出し。

### 画像書き出し

- 🖼️ **SVG**：ベクター、無限拡大可、再編集可
- 📷 **PNG**：2× Retina 解像度

どちらもノード、エッジ、色、境界、リレーション、サマリーを含みます。

## 自動保存

ブラウザ版：変更ごとに `localStorage` へ書き込み、リロード後も維持。

デスクトップ版：同様。データは `~/Library/Application Support/Agentic Mindmap/Local Storage/` に保存されます。明示的にファイルへ書き出したい場合は `⌘S` で JSON エクスポート。

## リセット / クリア

- ツールバー ✨ 新規：現在のマップをクリア（確認ダイアログ付き）
- ツールバー ⟲：選択ノードの自由位置 / サイズ / 拡大率をリセット
- 完全消去：デスクトップは `rm -rf "~/Library/Application Support/Agentic Mindmap"`、ブラウザは DevTools → Application → Local Storage をクリア

## 既知の制約

- Markdown インポートは未知の構文を無視します
- SVG / PNG 書き出しに絵文字フォントは埋め込まれません（システムフォントのレンダリングに依存）
- 初回 `npm run dist` で Electron ビルド資産（~150 MB）をダウンロードするため時間がかかります
- `npm run dist` の DMG は未署名です。初回起動は `xattr -cr <app>` か右クリック → 開く で許可してください
